# 🤖 AI 智能助手

MagicWord 集成了先进的大语言模型 (LLM)，让背词不再枯燥。

## 🔍 单词深度解析

在单词详情页，如果你觉得释义不够详细，可以点击 **“AI 解析”**。
AI 将为你生成：
1.  **助记法**：有趣的谐音、词根词缀记忆法。
2.  **场景例句**：地道的英文例句及翻译。
3.  **详细释义**：多重词性的完整解释。

## 📝 AI 批量导入 (Magic Import)

这是 MagicWord 最酷的功能之一！
你正在读一篇英语文章，想把里面的生词全部存下来？

1.  复制文章段落。
2.  在 App 中点击 **“导入” -> “AI 批量导入”**。
3.  粘贴文本。
4.  **AI 自动提取**：App 会自动识别文本中的单词和**词组**（如 `give up`），过滤掉简单词（the, is），并自动生成释义、音标和例句。
5.  一键保存到词库。

::: warning 注意
AI 提取可能需要几秒钟到一分钟，请保持网络连接畅通。
:::
