# AI 辅助功能

MagicWord 深度集成了生成式 AI，彻底告别手动输入释义的繁琐。

## AI 配置

在开始使用 AI 功能前，请确保已正确配置 API。
前往 **设置 (Settings)** 页面：

1.  **API 地址 (Base URL)**: 输入兼容 OpenAI 格式的 API 地址。
    *   默认: `https://api.openai.com/v1` (或您的中转服务地址)
2.  **API 密钥 (API Key)**: 输入您的 API Key (`sk-...`)。
3.  **模型名称 (Model)**: 输入您想使用的模型名称，例如 `gpt-3.5-turbo`, `gpt-4o`, 或 `gemini-pro` (取决于您的服务商支持)。

## 功能应用

### 1. 自动录入 (Auto Entry)
*   **触发**: 在主界面搜索框输入一个新单词，按回车。
*   **流程**: 若本地无此单词，APP 会向 AI 发送指令，要求生成该单词的 JSON 数据（含释义、音标、例句）。
*   **结果**: 生成成功后，单词自动存入数据库并显示。

### 2. 批量导入 (Bulk Import)
*   **触发**: 列表模式 -> 导入按钮。
*   **流程**: 将您输入的单词列表一次性发送给 AI。
*   **优势**: 相比逐个录入，批量导入效率更高，适合从文章或教材中整理单词表。

### 3. 智能完善
*   即便您只输入了单词拼写，AI 也会自动补全：
    *   **地道的中文释义**: 结合语境的解释。
    *   **助记方法**: 提供词根词缀分析或联想记忆法。
    *   **实用例句**: 展示单词在句子中的实际用法。

## 常见问题

*   **生成失败**: 请检查网络连接及 API Key 余额。
*   **响应慢**: 取决于您使用的模型服务商及网络状况。建议使用响应速度较快的模型（如 `gpt-3.5-turbo` 或专门的 Flash 模型）。
