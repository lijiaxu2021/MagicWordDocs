# ☁️ 在线词库共享

MagicWord 拥有一个开放的词库生态。

## 🌐 架构原理
我们的在线词库是 **去中心化** 的：
*   **数据源**：所有词库文件存储在 GitHub 上的公开仓库 `magicwordfile`。
*   **加速**：通过 Cloudflare Worker 全球 CDN 加速，解决 GitHub 访问慢的问题。
*   **索引**：GitHub Actions 自动运行，每当有新提交时自动更新索引文件。

## 🤝 参与贡献
你可以通过 App 直接上传你的优质词库。
1.  整理好你的本地词库。
2.  上传时添加准确的 **标签 (Tags)**（例如 `考研`, `计算机`, `医学`）。
3.  上传成功后，全球用户都能在几分钟内搜到你的词库！
