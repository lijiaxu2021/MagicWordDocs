# 常见问题 (FAQ)

## ❓ 安装与更新

**Q: 为什么安装包下载后无法安装？**
A: 请确保在手机设置中允许 App **“安装未知来源应用”**。Android 8.0+ 系统会在安装时自动请求此权限，请点击允许。

**Q: App 会自动更新吗？**
A: 是的。每次启动时 App 会检查 GitHub 的最新版本。如果有新版本，会弹窗提示更新。

## ❓ 使用问题

**Q: 在线词库加载很慢怎么办？**
A: 请检查你的网络连接。虽然我们使用了 CDN 加速，但在某些网络环境下仍可能受限。尝试切换 Wi-Fi 或数据流量。

**Q: AI 功能需要付费吗？**
A: 目前版本集成了开发者的 API Key，免费供大家体验。未来可能会支持用户自定义 API Key（如 OpenAI / DeepSeek / SiliconFlow）。

**Q: 数据会丢失吗？**
A: 数据存储在本地数据库中。建议定期通过 **“导出词库”** 功能备份重要数据。卸载 App 会清除本地数据！
