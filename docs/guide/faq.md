# 设置与常见问题

## 设置选项

点击主界面右上角的 **齿轮图标** 进入设置页面。

### AI 设置
*   **API Key**: 必填，用于调用 AI 生成释义。
*   **Base URL**: 必填，AI 服务接口地址。
*   **Model**: 指定使用的模型（如 `gpt-4o-mini`）。

### 个性化
*   **人设 (Persona)**: 您可以自定义 AI 的语气。例如：“你是一个严厉的英语老师”或“你是一个幽默的单词助手”。这会影响 AI 生成例句和记忆方法的风格。

### 其他
*   **自动更新**: 开启后，应用会自动检查 GitHub Release 是否有新版本。
*   **显示日志**: 开发调试用，普通用户建议关闭。

## 常见问题 (FAQ)

### Q: 为什么搜索单词没有反应？
A: 
1. 检查网络是否连接。
2. 检查 **设置** 中的 API Key 和 Base URL 是否正确。
3. 如果是新单词，AI 生成需要时间（通常 3-10 秒），请耐心等待加载转圈结束。

### Q: 在线词库下载失败？
A: 请确保您的网络可以访问 GitHub (Raw Content)。如果网络受限，可能无法下载词库文件。

### Q: 如何备份我的数据？
A: 
1. 使用 **导出** 功能将重要词库导出为 JSON 文件。
2. 即使应用卸载，重新安装后导入 JSON 即可恢复。
3. *注：应用数据存储在私有目录，清除应用数据会丢失未导出的词库。*

### Q: 为什么没有发音？
A: MagicWord 使用系统自带的 TTS (Text-to-Speech) 引擎。
*   请检查手机 **设置 > 系统 > 语言和输入法 > 文字转语音输出**。
*   确保已安装 Google TTS 或其他语音引擎包。

### Q: 导入的单词显示乱码？
A: 请确保导入的文本或文件是标准的 UTF-8 编码。
