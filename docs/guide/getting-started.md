# 快速开始

欢迎使用 MagicWord ！这是一个结合了 AI 辅助记忆和科学复习算法 (SM-2) 的单词学习应用。

## 1. 安装与初始化

1.  **下载安装包**: 从发布页面下载最新的 APK 文件。or 如果你在国内 那么可以进入./https://mag.upxuu.com/latest/Mag/ 下载最新版本
2.  **安装应用**:
    *   如果系统提示“禁止安装未知来源应用”，请前往 **设置 > 安全 > 安装未知应用**，允许浏览器或文件管理器安装应用。
    *   **权限授予**: 首次启动时，应用可能会请求存储权限（用于备份/恢复数据）和通知权限（用于复习提醒）。请务必授予 **存储权限**，否则无法正常读取本地词库。

## 2. 界面概览

应用采用简洁的卡片式设计，主要分为以下几个区域：

*   **顶部栏 (Top Bar)**:
    *   **左侧**: 侧边栏菜单/词库切换。点击标题或菜单图标可快速切换当前词库。
    *   **中间**: 当前词库名称。
    *   **右侧**: 视图切换（卡片/列表）、个人中心、设置。
*   **搜索/输入栏**: 位于顶部栏下方。
    *   **功能**: 支持全局搜索单词。
    *   **AI 录入**: 输入一个新单词并按下回车 (Enter)，若本地词库中不存在，AI 将自动联网生成释义并添加到当前词库。
*   **主内容区**:
    *   **卡片模式 (Card Mode)**: 类似闪卡，左右滑动切换单词，点击翻转查看释义。
    *   **列表模式 (List Mode)**: 显示单词列表，支持批量操作、排序和拖拽排序。

## 3. 您的第一步

1.  **添加词库**:
    *   应用预置了“默认词库”。
    *   点击左上角菜单 -> “管理词库” -> 点击 “+” 号 -> 输入名称即可创建新词库。
    *   或者，在“在线词库”中下载现成的词库（详见 [在线词库](./online.md)）。

2.  **录入单词**:
    *   在搜索框输入 `apple`，点击键盘上的搜索/回车键。
    *   稍等片刻，AI 会自动生成释义、音标和例句。
    *   单词卡片将立即出现在屏幕上。

3.  **开始学习**:
    *   点击卡片查看释义。
    *   根据记忆情况选择评分（0-5分），系统将自动安排下次复习时间。
