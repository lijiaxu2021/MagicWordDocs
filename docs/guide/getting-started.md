# 简介与安装

欢迎使用 **MagicWord**！这是一个集成了 AI、记忆算法和云端共享功能的现代化背词 App。

::: tip 提示
本项目完全开源，包含 Android 客户端、Cloudflare 服务端和数据仓库。
:::

## ✨ 主要特性

*   **📚 词库管理**：支持本地创建、导入导出、云端下载海量词库。
*   **🧠 记忆算法**：基于 SM-2 算法，科学安排复习时间。
*   **🤖 AI 助手**：一键解析单词，生成助记法和例句；支持 AI 批量从文本中提取单词。
*   **☁️ 云端同步**：词库和学习进度本地存储，支持导出备份；在线词库社区化运营。
*   **🎨 现代 UI**：Material You 设计，支持深色模式。

## 📥 下载安装

目前 MagicWord 支持 Android 平台。

1.  **GitHub Release**: 前往 [Releases 页面](https://github.com/lijiaxu2021/MagicWord/releases) 下载最新 APK。
2.  **自动更新**: 安装后，App 会自动检查更新并提示升级。

## 🚀 快速上手

1.  打开 App，你将看到简洁的主界面。
2.  点击左上角菜单，进入 **“词库管理”**。
3.  点击 **“在线词库”**，浏览并下载你感兴趣的词库（如“雅思核心词汇”）。
4.  下载完成后，回到主页，即可开始背词！
