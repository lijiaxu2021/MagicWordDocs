# 词库管理

MagicWord 提供了灵活的词库管理功能，支持本地创建、在线下载以及多种导入方式。

## 本地词库管理

点击主界面左上角的菜单图标或词库名称，进入 **词库管理** 界面。

### 1. 创建与切换
*   **创建**: 点击右下角的 **+** 悬浮按钮，输入名称即可创建一个空白词库。
*   **切换**: 在列表中点击任意词库，即可将其设置为“当前词库”，主界面将显示该词库的内容。

### 2. 编辑与删除
*   **重命名**: 长按词库项，选择“重命名”。
*   **删除**: 长按词库项，选择“删除”。**注意：删除操作不可恢复，该词库下的所有单词及学习记录都将被清空。**
*   **导出**: 长按词库项，选择“导出”，可将词库导出为 JSON 文件分享给他人或备份。

## 单词导入

在主界面（列表模式下），您可以批量导入单词。

### 1. 列表模式
点击右上角的 **列表图标** 切换到列表视图。此时右下角会出现 **导入** 按钮。

### 2. 批量导入 (Bulk Import)
点击 **导入** 按钮，打开导入面板：
*   **格式**: 支持多行输入，每行一个单词；或者使用逗号分隔。
    *   例如：
        ```text
        apple
        banana
        orange
        ```
        或者 `apple, banana, orange`
*   **AI 处理**: 点击“开始导入”后，应用会调用配置好的 AI 模型，依次为这些单词生成：
    *   中文释义
    *   英文释义
    *   音标
    *   例句
    *   记忆方法
*   **实时日志**: 面板下方会显示导入进度和结果。

### 3. 单词编辑与排序
*   **排序**: 在列表模式下，点击右上角的排序图标，支持按：
    *   创建时间 (新->旧 / 旧->新)
    *   字母顺序 (A->Z)
    *   复习次数 (高->低)
*   **拖拽排序**: 长按列表项右侧的拖拽手柄 (≡)，可以手动调整单词顺序。
*   **编辑**: 点击列表中的单词，或在卡片模式下点击“编辑”按钮，可修改释义、例句和记忆方法。

## 词库数据结构 (高级)

MagicWord 使用标准化的 JSON 格式存储词库，这使得词库非常容易分享和迁移。

### 词库文件结构
一个完整的词库包通常包含两个文件（在在线词库中）：
1.  `info.json`: 元数据（名称、描述、作者、标签）。
2.  `library.json`: 实际的单词数据。

本地导出的文件是一个包含所有信息的单一 JSON 文件，可以直接在其他设备上导入。

### 仓库即数据库 (Repository as Database)
我们的在线词库系统采用了创新的“仓库即数据库”设计：
*   所有词库存储在 GitHub 仓库中。
*   通过 GitHub Actions 自动生成索引。
*   通过 Cloudflare CDN 全球分发。
这意味着您贡献的词库将永久保存且免费可访问。
