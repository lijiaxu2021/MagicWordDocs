# 📚 词库管理

MagicWord 的核心是**词库 (Library)**。你可以拥有多个词库，并自由切换。

## 📂 本地词库

在 **“词库管理”** 界面，你可以看到所有已下载或创建的词库。

*   **切换词库**：点击词库卡片即可切换为当前学习词库。
*   **新建词库**：点击右下角 `+` 号，输入名称即可创建空词库。
*   **删除/重命名**：长按词库卡片，可以进行重命名或删除操作。

## ☁️ 在线词库

点击顶部的 **“在线词库”** 标签，进入云端世界。

*   **浏览**：列表展示了社区分享的热门词库。
*   **搜索**：
    *   **关键词**：在顶部搜索框输入名称（如“考研”）。
    *   **标签筛选**：点击左侧的 **“标签”** 图标，选择分类（如 `雅思`, `托福`, `编程`）。
*   **下载**：点击下载图标，稍等片刻即可导入本地。

::: info 技术细节
在线词库基于 GitHub 仓库存储，通过 Cloudflare Worker 加速分发，确保全球访问速度。
:::

## 📤 导入与导出

### 导入
除了从在线词库下载，你还可以：
1.  **JSON 导入**：支持标准 JSON 格式的词库文件。
2.  **AI 批量导入**：在主页输入一段英文文本，AI 会自动提取其中的单词并生成释义（详见 [AI 功能](/guide/ai)）。

### 导出与分享
想把你的词库分享给朋友？
1.  在本地词库列表中，点击 **“分享/上传”** 图标。
2.  **导出文件**：生成 JSON 文件，可以通过微信/QQ 发送。
3.  **上传到云端**：填写描述和标签，直接发布到在线词库，让所有人都能看到！
