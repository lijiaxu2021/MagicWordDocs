# 🧠 记忆与复习

MagicWord 内置了强大的 **SuperMemo-2 (SM-2)** 记忆算法，帮助你通过间隔重复 (Spaced Repetition) 高效记忆。

## 📖 学习模式

在主页，你会看到当前的单词卡片。

*   **卡片模式**：展示单词、音标。点击卡片查看释义。
*   **列表模式**：点击右上角切换，以列表形式浏览所有单词。

### 评分系统
当你查看单词释义后，需要根据记忆情况进行评分（0-5分）：

*   **0-2分 (忘/难)**：不认识或记错了。App 会在短时间内再次安排复习。
*   **3-5分 (会/熟)**：认识。App 会根据算法计算下一次复习时间（可能是1天后、6天后...）。

## 🔄 复习机制

App 会自动计算每个单词的 **“下一次复习时间”**。
*   **今日待复习**：主页只会优先显示今天需要复习的单词。
*   **新词学习**：如果今日任务完成，可以继续学习新录入的单词。

::: tip 小技巧
坚持每天打开 App 清理“待复习”列表，是记忆效果最好的方式！
:::

## 🔊 发音 (TTS)
点击卡片上的 🔊 图标，App 会调用系统的 TTS (Text-To-Speech) 引擎朗读单词和例句。确保你的手机已安装 Google TTS 或其他语音引擎。
