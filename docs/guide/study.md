# 记忆与复习 (Study)

MagicWord 的核心功能是基于 **SM-2 (SuperMemo-2) 间隔重复算法** 的智能复习系统。该系统根据您的记忆情况，自动安排单词的下一次复习时间，确保您以最高效的方式将短期记忆转化为长期记忆。

## 复习界面 (Study Screen)

点击底部导航栏的 **Study** 图标进入复习界面。

### 1. 自动开始
只要有单词需要复习（即 `下次复习时间 <= 今天`），进入界面后会自动开始复习会话。无需手动点击“开始”。

### 2. 卡片交互
复习过程采用经典的“闪卡 (Flashcard)”模式：
1.  **问题面**: 显示单词拼写、音标。
    *   *操作*: 点击卡片任意位置查看答案。
    *   *朗读*: 点击右上角的喇叭图标朗读单词。
2.  **答案面**: 显示中文释义、例句、记忆方法。
    *   *操作*: 根据您的记忆情况，点击底部的四个反馈按钮之一。

### 3. 反馈按钮与算法
您的反馈直接决定了单词下一次出现的时间。MagicWord 严格遵循 SM-2 算法逻辑：

| 按钮 (Button) | 颜色 | 含义 | 算法行为 (Algorithm Effect) |
| :--- | :--- | :--- | :--- |
| **忘记 (Forgot)** | 🔴 红色 | 完全没印象 | **重置进度**: 连对次数归零，间隔重置为 1 天。 |
| **困难 (Hard)** | 🟠 橙色 | 犹豫很久才想起来 | **保持或微增**: 间隔正常增长，但难度系数 (EF) 降低。 |
| **良好 (Good)** | 🟢 绿色 | 稍微想了一下 | **正常增长**: 间隔正常增长，难度系数不变。 |
| **简单 (Easy)** | 🔵 蓝色 | 秒答 | **加速增长**: 间隔大幅增长，难度系数 (EF) 增加。 |

### 4. 进度条
界面顶部的线性进度条显示当前复习队列的完成进度。

## SM-2 算法详解 (高级)

MagicWord 实现的 SM-2 算法维护了每个单词的三个核心属性：

1.  **Repetitions (连对次数)**: 连续选择“困难”及以上评价的次数。
2.  **Interval (间隔天数)**: 下一次复习在几天后。
3.  **Easiness Factor (EF, 难度系数)**: 衡量单词对您来说有多容易。初始值为 2.5，范围 1.3 ~ 3.0+。

### 增长公式
*   第 1 次复习: 间隔 1 天
*   第 2 次复习: 间隔 6 天
*   第 n 次复习: `前一次间隔 * EF`

例如，一个单词如果您一直选“简单”(EF 增加)，它的复习路径可能是：
`1天 -> 6天 -> 15天 -> 38天 -> 95天...`

如果您选“忘记”，路径会重置：
`... -> 38天 -> (忘记) -> 1天 -> 6天...`

## 多词库复习

MagicWord 支持同时复习多个词库：
1.  点击复习界面右上角的 **设置/筛选图标**。
2.  在弹出的对话框中勾选您想一起复习的词库。
3.  系统会自动聚合所有选中词库中“到期”的单词，打乱顺序呈现给您。

## 统计信息
在卡片的答案面，您可以查看到该单词的详细统计：
*   **学习次数**: 总共复习了多少次。
*   **正确率**: 选择“困难/良好/简单”的比例。
*   **下次复习**: 根据本次反馈计算出的具体日期。
*   **上次复习**: 上次复习的日期。
