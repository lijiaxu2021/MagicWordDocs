# 测试模式 (Test Mode)

除了日常的卡片复习，您还可以通过测试模式来检验学习成果。

## 进入测试

1.  在 **列表模式** 下。
2.  **长按** 选中一个或多个单词（或点击右上角“全选”）。
3.  点击左上角的 **播放 (Play)** 图标。
4.  在弹出的对话框中选择测试类型。

## 测试类型

### 1. 选择题模式 (Choice Mode)
*   **形式**: 屏幕显示单词，下方给出 4 个中文释义选项。
*   **规则**:
    *   选对：显示绿色，自动进入下一题。
    *   选错：显示红色，并高亮正确答案，需手动点击下一题。
*   **计分**: 答对得 1 分，答错不得分。

### 2. 拼写模式 (Spell Mode)
*   **形式**: 屏幕显示中文释义，隐藏单词拼写。
*   **操作**: 在输入框中输入单词拼写，点击确认。
*   **规则**:
    *   拼写完全正确才算通过。
    *   适合深度记忆和检查拼写准确性。

## 测试结果

测试结束后，系统会显示：
*   **总分**: 本次测试的得分。
*   **正确率**: 答对题数 / 总题数。
*   **错题回顾**: (后续版本功能) 建议针对错题进行针对性复习。
