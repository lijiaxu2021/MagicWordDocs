# 📝 测试与统计

检验学习成果的最佳方式就是测试。

## 🧪 测试模式

在单词列表页面，选中一组单词，点击 **“测试”** 图标。

### 1. 选择题模式 (Choice)
*   给出单词，从4个中文选项中选择正确的一个。
*   快速刷词，适合初记。

### 2. 拼写模式 (Spell)
*   给出中文释义，需要手动拼写出英文单词。
*   深度记忆，适合复习。

## 📊 错题本
测试中答错的单词会自动进入 **“错题本”**，并降低其 SM-2 评分（视为“忘记”），增加未来的复习频率。

## 📈 学习统计
在 **“个人中心”**，你可以查看：
*   累计学习天数
*   已掌握单词数
*   今日复习进度
