# 设置与配置 (Settings)

**SettingsScreen** 是管理 MagicWord 全局行为的控制中心。在这里，您可以配置 AI、个性化选项、系统更新以及查看日志。

## 1. AI 配置 (AI Configuration)

这是 App 最核心的配置区。

### Key-Kit 验证
*   **输入框**: 输入管理员提供的 Key (如 `vip-001`)。
*   **验证**: 点击按钮后，App 会联网验证。成功后自动填入 API Key 和 Model Name。
*   **重置**: 如果您想切换 Key，直接输入新 Key 并再次验证即可。

### 手动配置
*   **API Key**: 只读显示（如果是通过 Key-Kit 获取的）。如果是手动模式，可以输入您的 `sk-...` 密钥。
*   **Model Name**: 默认为 `Qwen/Qwen2.5-7B-Instruct`。您可以修改为其他兼容 OpenAI 格式的模型。

## 2. 个性化 (Personalization)

### User Persona (用户画像)
告诉 AI 您是谁，以便生成更贴切的例句。
*   *示例*: "我是初中生" -> AI 会生成简单词汇的例句。
*   *示例*: "我是托福考生" -> AI 会生成学术场景的例句。
*   *示例*: "我是哈利波特迷" -> AI 会尝试生成与魔法世界相关的例句。

### Save Location (默认存储位置)
决定新录入的单词存放在哪个词库。
*   **跟随当前 (Default)**: 存入当前在主界面选中的词库。
*   **指定词库**: 强制存入某个特定词库（例如 "生词本"），无论当前在浏览哪个词库。

## 3. 系统偏好 (Preferences)

*   **Enable Logging**: 开启调试日志。开启后，App 会记录详细的运行信息到本地文件。建议仅在遇到 Bug 需要反馈时开启。
*   **Auto Update**: 开启自动更新检查。建议保持开启，以便第一时间获取新功能。

## 4. 更新与支持

*   **检查更新**: 手动检查新版本。如果有更新，会弹出对话框提示下载。
*   **关于与支持**:
    *   **使用文档**: 跳转到本在线文档。
    *   **GitHub 仓库**: 查看源码。
    *   **开发者博客**: 访问 UpXuu.com。

## 5. 日志管理 (Log Management)

点击 **查看日志** 进入日志列表界面。
*   **列表**: 按时间倒序显示所有日志文件。
*   **详情**: 点击可查看具体日志内容。
*   **分享**: 点击分享图标，可以将日志文件发送给开发者进行排查。
